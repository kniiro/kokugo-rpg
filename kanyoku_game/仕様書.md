# 【スマホ向け】国語克服バトルRPG 仕様書 (v1.3)

## 1. プロジェクト概要
小学5年生向けの国語（慣用句、ことわざ、類義語・対義語、文法、四字熟語、主語・述語）学習用ブラウザゲーム。
ポケモン風のRPG要素とすごろくを掛け合わせたシステム。

### 技術スタック
- フロントエンド: HTML5 / CSS3 / JavaScript (Vanilla JS)
- データ形式: CSV (`questions.csv`)
- PWA対応: iOSのホーム画面追加時にフルスクリーン表示されるよう最適化。

## 3. アセット仕様
### 画像
- **3Dスプライト**: 敵1〜4はAI生成による高品質3Dモデル。背景は完全透過。
- **キャッシュ対策**: 画像読み込み時にクエリパラメータ（`?v=timestamp`）を付与し、更新が即時反映されるように設計。
- **アプリアイコン**: `images/enemy9.png` を `apple-touch-icon` として利用。

### 音声 (`audio/`)
- `masala.mp3`: マップ移動BGM
- `battle_start.mp3`: バトル開始SE
- `battle_normal.mp3`: 通常バトルBGM
- `battle_boss.mp3`: 中ボスBGM (カビゴン以降)
- `battle_last.mp3`: ラスボスBGM (ミュウツー)
- `click.mp3`: UI操作音

## 4. ゲームシステム
### 基本フロー
- 全10ステージを攻略。クリアすると次のステージが解放。
- **再プレイ機能**: クリア済みのステージをタップして再挑戦可能。
- **捕獲システム**: 敵撃破後にモンスターボールを投げる物理シミュレーションモード。
- **捕獲制限**: 同種のポケモンは最大2匹までしかストックできない。

### バトルロジック
- **HPシステム**: 敵HP（必要正答数）とプレイヤーのミス上限を設定。
- **身代わり**: プレイヤーHPが0になった際、仲間のポケモンが1体「みがわり」となり、HP1で継続可能。

## 5. 出題システム
### カテゴリー
- 慣用句、ことわざ、類義語・対義語、文法、四字熟語、主語・述語

### 難易度レベル (1〜5)
- ステージごとに定義されたレベル範囲から問題を抽出。
- **四字熟語**: ID 361〜380は中学受験レベル（小学生向け高難易度）に特化。
- **均等配分**: ID 301〜360（主語・述語、類義語・対義語）はレベル2〜5に均等に出現するよう調整。
